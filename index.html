OVERVIEW

연구 개요

도시 철도망 설계를 순수 수학·알고리즘으로 접근합니다. 서로 다른 위상학적 방법론을 가진 세 가지 알고리즘을 서울시 403개 역 데이터와 Mumford3 벤치마크 네트워크에 적용하여 비교 분석합니다.

$TL$ 총 길이 $TL = \sum_{e \in E} l_e$ — 경제성, 낮을수록 우수
$WMD$ 가중 평균 이동 거리 $WMD = \dfrac{\sum_{u \neq v} d_{uv} w_{uv}}{\sum_{u \neq v} w_{uv}}$ — 효율성
$FT$ 고장 허용성 $FT = \dfrac{|E| - |B|}{|E|}$ — 신뢰성, 높을수록 우수
$\lambda_2$ 대수적 연결성 라플라시안 $L$의 두 번째 고윳값 — 전역 강건성

ALGORITHM 01

점균류 알고리즘

황색망사점균(Physarum polycephalum)의 먹이 탐지 행동을 Hagen–Poiseuille 법칙으로 수치화합니다. 선행 연구(Tero et al., 2010)에서 36개 역에 적용하여 도쿄 철도망과 유사한 네트워크를 자생적으로 탐지함을 확인하였습니다. 본 연구에서는 서울시 313개 역 및 Mumford3 128개 역에 확장·적용하고 매개변수 $\gamma$에 따른 위상 변화를 수학적으로 분석합니다.

수학적 모델

델로네 삼각분할로 구성된 초기 메쉬에서 출발하여, 흐름이 많은 관은 확장하고 적은 관은 수축시키는 적응형 피드백을 반복합니다.

유량 방정식 (Hagen–Poiseuille)
$$Q_{ij} = D_{ij} \cdot \frac{p_i - p_j}{L_{ij}}$$
$D_{ij}$: 관 전도도, $p_i$: 노드 $i$에서의 압력, $L_{ij}$: 관의 길이
전도도 변화율 (적응형 피드백)
$$\frac{dD_{ij}}{dt} = \alpha|Q_{ij}|^\gamma - \mu D_{ij}$$
$\alpha=10.0$, $\mu=0.6$, $\gamma$: 핵심 비선형 제어 변수
각 노드 유량 보존 (키르히호프 법칙)
$$\sum_j Q_{ij} = S_i \quad \Rightarrow \quad Lp = S$$
$L$: 전도도 가중 라플라시안, 평균 승하차량을 $S_i$로 설정

핵심 발견 — $\gamma$에 따른 상전이

  • 저감마 $\gamma < 1.0$: 고밀도 네트워크, 높은 $FT \cdot \lambda_2$, 낮은 경제성
  • 최적 구간 $1.1 \lesssim \gamma \lesssim 1.4$: 경제성–신뢰성 균형점
  • 고감마 $\gamma > 1.6$: 트리 구조로 붕괴, $FT \cdot \lambda_2$ 급락

야코비안 행렬 분석으로 $\gamma = 1$ 임계값에서의 네트워크 붕괴 조건을 수학적으로 증명

네트워크 붕괴 수학적 증명 — 야코비안
$$\left.\frac{\partial \dot{D}_{23}}{\partial D_{23}}\right|_{D_{23}=0} = \alpha\gamma|Q_{23}|^{\gamma-1}\cdot\frac{\partial|Q_{23}|}{\partial D_{23}} - \mu$$
$\gamma \geq 1$이면 이 값 $< 0$ → $D_{23}=0$이 안정 평형점 → 간선 영구 소멸
$\gamma$ 값에 따른 서울시 지하철 노선도 변화 $\gamma=0.8$ (고밀도) → $1.3$ → $1.8$ → $2.3$ (트리 구조) — 실제 Matlab 시뮬레이션 결과
$\gamma$ 변화에 따른 효율성–신뢰성 트레이드오프 Mean Distance(MD) vs. Alpha Index($\alpha$): $\gamma$ 증가 → 연결 다양성($\alpha$) 감소, 효율성 변화
$\gamma=1.0$ (좌) / $1.6$ (중) / $2.2$ (우) 네트워크 밀도 비교 반복 횟수 250회 수렴 결과 — $\gamma$ 증가에 따른 간선 소실 확인
ALGORITHM 02

NSGA-2 다목적 유전 알고리즘

상충하는 세 목적함수 $TL$, $WMD$, $\lambda_2$를 동시에 만족하는 파레토 최적해 집합을 진화적으로 탐색합니다. 단일 최적해가 아닌 50개의 균형 잡힌 해 집합을 도출하여, 도시 계획가가 예산·안전 수준에 따라 의사결정할 수 있는 정량적 공간을 제공합니다.

수학적 모델

노선도를 후보 간선의 이진 염색체로 인코딩합니다. 교차·변이 연산을 통해 세대를 진화시키며, 비지배 정렬과 Crowding Distance로 파레토 프론트의 다양성을 유지합니다.

다목적 최적화 문제
$$\min\,(TL,\; WMD), \quad \max\,\lambda_2$$
세 목적함수는 상충 관계(트레이드오프) — 단일 최적해 존재 불가
파레토 지배 조건
$$\mathbf{x} \succ \mathbf{y} \iff \forall i:\; f_i(\mathbf{x}) \leq f_i(\mathbf{y}) \;\wedge\; \exists j:\; f_j(\mathbf{x}) < f_j(\mathbf{y})$$
Crowding Distance (다양성 보존)
$$d_i = \sum_{m=1}^{M} \frac{\left|f_m(i+1) - f_m(i-1)\right|}{f_m^{\max} - f_m^{\min}}$$
$d_i$가 클수록 주변 밀집도 낮음 → 우선 선택하여 프론트 다양성 유지
대수적 연결성 $\lambda_2$ (목적함수)
$$L = D - A, \quad 0 = \lambda_1 \leq \lambda_2 \leq \cdots \leq \lambda_n$$
$\lambda_2 > 0$ iff 연결 그래프, 클수록 분리 저항 강함

핵심 발견 — 실제 서울의 위치

  • $TL$·$WMD$ 기준: 파레토 프론트에 근접 (준수한 효율성)
  • $\lambda_2$ 기준: 프론트 하방 위치 → 개선 여지 존재
  • 특정 환승역 집중으로 인한 구조적 취약점 확인

현행 서울 지하철은 수요 집중 구간에 과도하게 최적화되어 강건성 측면에서 한계

$TL$–$WMD$–$\lambda_2$ 3차원 파레토 최적해 (50개) ◆ 실제 서울 지하철 · ★ 호선 기반 알고리즘의 상대적 위치
파레토 프론트 2차원 투영 (4가지 관점) $TL$–$WMD$, $TL$–$\lambda_2$, $TL$–$FT$, $WMD$–$\lambda_2$
ALGORITHM 03

호선 기반 노선도 생성 알고리즘

결정론적 8단계 프로세스로 호선 구조를 명시적으로 생성합니다. 동일한 입력에 대해 항상 동일한 결과를 보장하며, 볼록 껍질의 재귀적 박피와 사잇각 판별을 통해 호선이라는 개념을 명시적으로 산출하는 최초의 위상 설계 알고리즘입니다.

핵심 수식

볼록 껍질(Quickhull)로 외곽부터 내부로 층별로 역사를 배정합니다. 각 층의 가장 먼 두 역을 종점으로 하는 가중치 $d^3$ 다익스트라 경로가 호선 뼈대가 됩니다.

층별 종점 선정 (볼록껍질)
$$(u^*, v^*) = \arg\max_{u,\,v\,\in\,\mathrm{CH}(S_k)}\;\|u - v\|$$
다익스트라 뼈대 — 초 장거리 페널티
$$w(u,v) = \|u - v\|^3$$
거리의 세제곱 가중치로 실제 철도의 초장거리 노선 부재를 반영
재귀적 분할 정복 봉합 — 수선의 발 투영
$$t = \frac{(\mathbf{w}-\mathbf{u}) \cdot (\mathbf{v}-\mathbf{u})}{\|\mathbf{v}-\mathbf{u}\|^2},\quad t \in [0,1]$$
선분 $uv$ 위의 수선의 발 $\mathbf{p} = \mathbf{u}+t(\mathbf{v}-\mathbf{u})$로 최적 징검다리 역 탐색
MST 환승 링크 (호선 간 통합)
$$T^* = \arg\min_{T \subseteq G_\mathcal{L}}\;\sum_{(i,j)\in T} \min_{u\in L_i,\,v\in L_j}\|u-v\|$$
지름길 선정 조건 ($\lambda=7$, 사잇각 $\theta$)
$$d_y \;\geq\; \lambda \cdot d_x \;\wedge\; \mathrm{hops} > 3$$
$\theta < 60°$: 본선 편입 (기존 간선 교체) · $\theta \geq 60°$: 지선 편입

핵심 발견 — 실제 서울 대비 성능

  • $FT$: $0.889 \to \mathbf{0.930}$ (고장 허용성 향상)
  • $Cent$: $0.180 \to \mathbf{0.089}$ (트래픽 집중도 50.5% 감소)
  • $TL$·$WMD$ 소폭 증가는 외곽 우회 경로 확보에 기인

볼록껍질 우회 경로가 강건성을 끌어올리는 직접적 원인 — 강남·잠실 등 핵심역 장애 시 전역 붕괴 방지

Mumford3 네트워크 단계별 진행 (127개 역, 12프레임) 볼록껍질 박피 8개 레이어 → 재귀 봉합 → MST → 지름길 편입. 논문 파라미터: $\Delta=6.48$, $\lambda=9$
볼록껍질 동심 레이어 구조 헐 경계(점선) · 다익스트라 경로(실선)의 계층적 구조 — 서울시 적용 결과
지름길 선정 통계 — $d_x$ vs $d_y$ 산점도 $\lambda=7$ 기준선과 본선/지선 지름길 분류 분포
매개중심성 상위 역 기준 홉 거리 등고선 지도 $Cent$ 감소의 시각적 근거 — 트래픽 분산 효과 확인

CONCLUSION

결론 및 고찰

🧫

점균류 알고리즘

$FT$·$\lambda_2$ 측면에서 가장 우수하나 $TL$이 실제 대비 10배 이상으로 경제성이 낮습니다. 수학적 분석으로 $\gamma=1$ 임계값에서 네트워크 붕괴가 발생함을 야코비안 행렬로 증명했습니다. 실제 데이터에서의 임계값 편차는 간선 수에 따른 상호작용 지연으로 해석됩니다.

🧬

NSGA-2

$TL$·$WMD$·$\lambda_2$ 상충 관계를 파레토 최적해 집합으로 균형화합니다. 실제 서울을 투영하면 효율성은 준수하나 $\lambda_2$ 개선 여지가 확인됩니다. 예산·안전 수준에 맞는 정량적 의사결정 공간을 제공합니다.

🗺️

호선 기반 알고리즘

결정론적 설계로 재현성을 보장합니다. $TL$·$WMD$ 소폭 증가에도 $FT$ 향상 + $Cent$ 50% 감소를 달성합니다. $\lambda$ 조정으로 경제성–신뢰성 맞춤 설계가 가능하며, 세 알고리즘 중 유일하게 실제 노선도 형태로 결과를 직접 산출합니다.
