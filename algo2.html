ALGORITHM 02

NSGA-2 다목적 유전 알고리즘

상충하는 세 목적함수 $TL$, $WMD$, $\lambda_2$를 동시에 만족하는 파레토 최적해 집합을 진화적으로 탐색합니다. 단일 최적해가 아닌 50개의 균형 잡힌 해 집합을 도출하여, 도시 계획가가 예산·안전 수준에 따라 의사결정할 수 있는 정량적 공간을 제공합니다.

수학적 모델

노선도를 후보 간선의 이진 염색체로 인코딩합니다. 교차·변이 연산을 통해 세대를 진화시키며, 비지배 정렬과 Crowding Distance로 파레토 프론트의 다양성을 유지합니다.

다목적 최적화 문제
$$\min\,(TL,\; WMD), \quad \max\,\lambda_2$$
세 목적함수는 상충 관계(트레이드오프) — 단일 최적해 존재 불가
파레토 지배 조건
$$\mathbf{x} \succ \mathbf{y} \iff \forall i:\; f_i(\mathbf{x}) \leq f_i(\mathbf{y}) \;\wedge\; \exists j:\; f_j(\mathbf{x}) < f_j(\mathbf{y})$$
Crowding Distance (다양성 보존)
$$d_i = \sum_{m=1}^{M} \frac{\left|f_m(i+1) - f_m(i-1)\right|}{f_m^{\max} - f_m^{\min}}$$
$d_i$가 클수록 주변 밀집도 낮음 → 우선 선택하여 프론트 다양성 유지
대수적 연결성 $\lambda_2$ (목적함수)
$$L = D - A, \quad 0 = \lambda_1 \leq \lambda_2 \leq \cdots \leq \lambda_n$$
$\lambda_2 > 0$ iff 연결 그래프, 클수록 분리 저항 강함

핵심 발견 — 실제 서울의 위치

  • $TL$·$WMD$ 기준: 파레토 프론트에 근접 (준수한 효율성)
  • $\lambda_2$ 기준: 프론트 하방 위치 → 개선 여지 존재
  • 특정 환승역 집중으로 인한 구조적 취약점 확인

현행 서울 지하철은 수요 집중 구간에 과도하게 최적화되어 강건성 측면에서 한계

$TL$–$WMD$–$\lambda_2$ 3차원 파레토 최적해 (50개) ◆ 실제 서울 지하철 · ★ 호선 기반 알고리즘의 상대적 위치
파레토 프론트 2차원 투영 (4가지 관점) $TL$–$WMD$, $TL$–$\lambda_2$, $TL$–$FT$, $WMD$–$\lambda_2$
